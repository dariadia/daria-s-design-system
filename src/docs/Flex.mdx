# Flex

Flex is a flexbox block. [The Flexbox Layout (Flexible Box)](https://www.w3.org/TR/css-flexbox/) provides a more efficient way to lay out, align and distribute space among items in a container. 

The main idea is for the container to alter its itemsâ€™ width/height (and order) to best fill the available space (mostly to accommodate to all kind of display devices and screen sizes).
