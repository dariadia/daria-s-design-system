# Details

Details is a block based on the [HTML `<details />`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/details) element. It might have a summary and the main body, which is hidden under the cut unless clicked or specified accordingly.

## Properties

<Props of={Details} />

## Basic usage

<pre>
  <Details summary="Hello world in a box">
    <Box>Hello world! Read all about this</Box>
  </Details>
</pre>
