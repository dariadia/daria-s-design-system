# Button

Button is a box which one can click. It is rendered as `<button />` html element unless enforced differently. Buttons can be disabled if the user has to complete something first, e.g. fill out an input.

## Properties

<Props of={Button} />

## Basic usage

<pre>
  <Button onClick={() => alert('Hello World!')}>Click me!</Button>
</pre>
