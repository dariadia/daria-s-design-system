# Status Line

Status Line is a similarly-looking to an input range line. It has a dimmed background and a brighter filling of selected colour.

## Properties

<Props of={StatusLine} />

## Basic usage

<pre>
  <StatusLine
    accentColor="bgSecondaryDark"
    mainColor="bgDefault"
    percent={50}
  />
</pre>
