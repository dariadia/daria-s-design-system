# Grid

Grid Layout is a 2-dimensional system, meaning it can handle both columns and rows, unlike Flex, which is largely a 1-dimensional system.

<Props of={Grid} />

# Grid Kinds

## Grid (default)

<Grid
    mt={16}
    sx={{
      gridTemplateColumns: 'repeat(4, 1fr)',
      gridGap: 16,
    }}
  >
    <Box bg="accentLight" sx={{ borderRadius: "m" }} width="150px" p="xs" my="l" textAlign="center">Sample box</Box>
    <Box color="white" bg="accentDark" sx={{ borderRadius: "m" }} width="150px" p="xs" my="l" textAlign="center">Sample box</Box>
</Grid>
