# List

List is an `<ul>` element with the corresponding `<li>` children. Styles can be applied to the list as whole as well to the children separately.


## Properties

<Props of={List} />

## Basic usage

<List>
  <span>buy coffee</span>
  <span>cook coffee</span>
  <span>be happy</span>
</List>
