# Horizontal Scroller

Horizontal Scroller is a box (`<List />`) with children. One can scroll the elements inside the box as its overflow is hidden but horizontally scrollable.

## Properties

<Props of={HorizontalScroller} />

## Basic usage

<pre>
  <HorizontalScroller>
    <Box m="s" py="s" px="l" sx={{ border: '1px solid black'}}>Lorem ipsum</Box>
    <Box m="s" py="s" px="l" sx={{ border: '1px solid black'}}>Hello world</Box>
    <Box m="s" py="s" px="l" sx={{ border: '1px solid black'}}>Hallo</Box>
    <Box m="s" py="s" px="l" sx={{ border: '1px solid black'}}>Da bin ich</Box>
    <Box m="s" py="s" px="l" sx={{ border: '1px solid black'}}>Sample box</Box>
    <Box m="s" py="s" px="l" sx={{ border: '1px solid black'}}>Hello world</Box>
  </HorizontalScroller>
</pre>
